Manutenção do Sistema
Ferramentas de Manutenção: Use estas ferramentas para corrigir problemas de dados e normalizar informações no banco de dados.
Normalização de Tipos de Veículo e Carroceria

Esta ferramenta normaliza todos os tipos de veículo e carroceria no banco de dados, corrigindo variações de escrita como "caamba" para "cacamba", "CAÇAMBA" para "cacamba", etc.

Atenção: Esta operação atualizará todos os registros de motoristas. Certifique-se de que há um backup antes de executar.

Limpeza de Motoristas Duplicados

Esta ferramenta verifica e remove motoristas duplicados do banco de dados, mantendo apenas os registros com telefone no formato correto: 55+DDD+9+Número (ex: 5581986919496).

Atenção: Esta operação irá:

  • Identificar duplicados pelo telefone (telefone1)
  • Manter apenas registros com formatação correta (55+DDD+9+Número)
  • Remover duplicados com formatação incorreta
  • Se houver múltiplos com formatação correta, manterá o mais recente

Importante: Certifique-se de que há um backup antes de executar. Esta operação não pode ser desfeita.

Excluir Todo o Cadastro de Motoristas

⚠️ ATENÇÃO: OPERAÇÃO IRREVERSÍVEL
Esta ferramenta irá DELETAR TODOS OS MOTORISTAS do banco de dados. Use esta opção apenas quando desejar limpar completamente o cadastro para importar novos dados.

Esta operação irá:

  • Deletar TODOS os registros da tabela motoristas
  • Deletar TODOS os favoritos relacionados (tabela motoristas_favoritos)
  • Limpar completamente o cadastro
  • NÃO PODE SER DESFEITA

CRÍTICO: Faça um backup completo antes de executar. Após a exclusão, você precisará importar todos os dados novamente.
